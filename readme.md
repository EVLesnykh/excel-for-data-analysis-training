# Excel Data Analysis Training

Учебный репозиторий с практическими заданиями и работами по анализу данных в Excel.

## Описание
В этом репозитории собраны выполненные домашние задания и практические упражнения, полученные в рамках обучения по теме **анализ данных в Excel**.
Репозиторий представляет собой коллекцию учебных материалов, которые помогают закрепить навыки работы с таблицами, формулами, сводными таблицами и визуализацией данных.

## Структура репозитория
Проекты и задания организованы по лекциям и номерам домашних работ:

## Навыки, демонстрируемые в репозитории
- Формулы и функции Excel (SUM, AVERAGE, IF, VLOOKUP/XLOOKUP и др.)
- Сводные таблицы и аналитика
- Визуализация данных (диаграммы, графики)
- Подготовка и очистка данных
- Применение знаний из лекций на практике
